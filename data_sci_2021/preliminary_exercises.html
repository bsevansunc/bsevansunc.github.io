Preliminary exercises

We ask that, regardless of your current level of R knowledge, you please complete the following preliminary exercises prior to attending the workshop. Please feel free to email Brian Evans with any questions at evansbr@si.edu


  • Introduction to R Studio. This lesson will provide new R users with the baseline knowledge necessary to use R and R Studio and align experienced R users with some of the terminology and best management practices used throughout this workshop.

  • Lesson 1: Values. Learn the primary classes of values you will use in this workshop

  • Lesson 2: Objects. Learn the primary types of objects that are used to store values

  • Lesson 3: Indexing. Learn how to query objects using indexing in base R

  • Lesson 4: Functions. Learn how to write custom functions

Do you want to do more workshop preparation or practice your R skills? If so, we strongly recommend completing these SWIRL workshops prior to the workshop.
